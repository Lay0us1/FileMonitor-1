# FileMonitor

* FileMonitor文件监控小工具，监控文件，新增，修改，删除，重命名。 (如果感觉对您有帮助，感觉不错的话，请您给个大大的 ⭐️❗️)
* 用于代码审计时监控文件夹内文件的变动。
* 第一次运行时请根据提示在此页面下载对应动态依赖库放到指定位置即可（注意版本！）。

**V1.0**
```
java -jar FileMonitor.jar -p 监控目录
```
